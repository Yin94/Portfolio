Projects

Old Chengdu

A web app for one of my favorite local Chinese restaurant to allow their customers to view menu and make orders online. Frontend was built using html, javascript, CSS3 together with ReactJs library, Sass CSS preprocessor and Redux state management tool. Initial backend was google Firebase together with Alogolia for instant and full text search. And was replaced by a NodeJs Express server and mongoDB as backend.

Ez Mart

An end-to-end second hand products trading system on which users can browse, mark and buy items other users posted. Making use of CraigsList core utilities as reference, the system is equipped with the functionalities such as making a selling post, buying item, contacting seller and saving items for later thoughts, as well as some other functions such as item viewing modals.
